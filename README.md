# cool-game
1. If you haven't installed Cargo and Rust:

        $ curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

2. Clone:

       $ git clone https://github.com/SamaelCH/cool-game

3. Build:

       $ cd cool-game
       cargo build

4. Run:

       $ cargo run
      or:
        
       $ ./target/debug/cool-game

5. Enjoy

documentation can be found at CG.odt
